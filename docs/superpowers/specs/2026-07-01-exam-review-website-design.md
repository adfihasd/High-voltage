**状态**: 已确认

## 1. 项目目标

将高电压期末考试范围内容制作成一个单页面复习网站（纯静态 HTML），支持按章节浏览、知识点滑动切换、相关知识点跳转、以及基于分数分布的题目预测。

## 2. 技术方案

- **纯静态 HTML 单文件**：CSS + JS 全部内联，零依赖，双击即可在浏览器打开
- **交互方式**：手机触屏滑动 + 左右箭头按钮 + 键盘左右方向键
- **视觉风格**：学术简约，白底黑字，衬线字体，米白护眼背景

## 5. 交互设计

### 5.1 知识点切换
- **触屏滑动**：touchstart/touchend，滑动距离 > 50px 触发切换，CSS transition 动画（300ms）
- **箭头按钮**：固定于内容区左右两侧，桌面端可见，手机端半透明
- **键盘**：← → 方向键切换
- **章节边界**：
  - 当前章第一个知识点 + 左滑/左箭头 → 弹出确认浮层"是否回到第X章？"
  - 当前章最后一个知识点 + 右滑/右箭头 → 弹出确认浮层"是否进入第X章？"

### 5.2 章节选择
- 点击章节标签直接跳转到该章第一个知识点
- 当前章节高亮
- 横向可滚动（手机端友好）

### 5.3 相关链接跳转
- 点击相关知识链接 → 切换章节 + 定位到目标知识点
- 平滑过渡动画

### 6.2 题目数量分配规则

每章每种题型的题目数量按 `predictScore / totalPredictScore × 该题型总题数` 计算，结果四舍五入，至少1道。

**全局题型分布**（共11种题型，总题量基准）：
- 单选题（2分/道）：约10道 → 按分数占比分配
- 多选题（3分/道）：约10道 → 按分数占比分配
- 判断题（1分/道）：约10道 → 按分数占比分配
- 名词解释（3分/道）：约5道 → 按分数占比分配
- 简答题（5分/道）：约4道 → 按分数占比分配
- 论述题：每章至少1道（重点章节2道）
- 计算题（10分/道）：仅第7章1道、第8章1道

### 6.3 题型触发条件

- **定义型知识点** → 名词解释、判断、单选
- **机理/过程型知识点** → 简答、论述、多选
- **对比型知识点** → 简答、多选
- **数值/公式型知识点** → 单选、计算（如适用）

每个知识点列出所有可能题型，不重复列出无关题型。

## 8. 实现要点

1. 数据与UI分离：知识点数据用JS对象存储，UI通过模板渲染
2. 触屏手势：处理touchstart/touchend，计算deltaX判断方向，阻止页面默认滚动干扰
3. 动画过渡：CSS transform + opacity 过渡，左滑右滑方向视觉一致
4. URL hash：`index.html#1-3` 支持直接链接到特定知识点
5. 响应式设计：手机（<768px）全宽布局，桌面最大宽度900px居中
